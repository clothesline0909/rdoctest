    class Bakery::PackFactory

    This class represents a utility for creating Packs.

    Public Instance Methods

    build(pack_config) click to toggle source

    This method builds an array of packs from a pack_config array. The format of pack_config is:

    pack_config = [
  { code: 'CODE', price: 9.99, quantity: 5}
]

    This method raises an InvalidPackConfig exception if pack_config doesn't have the correct format.

    # File lib/bakery/pack_factory.rb, line 16
def build(pack_config)
  pack_config.map do |config|
    # Raise exception if invalid config is passed.
    raise Bakery::InvalidPackConfig unless valid_config?(config)

    build_pack(config)
  end
end

  Pack.new(

    price: config[:price],
    quantity: config[:quantity]

    valid_config?(config) click to toggle source

    This method checks that the config has a valid format. config must contain code, price and quantity keys.

    # File lib/bakery/pack_factory.rb, line 40
def valid_config?(config)
  # Get passed keys.
  keys = config.keys

  # Are all required keys present?
  %[code price quantity].all? { |key| keys.include?(key) }
end
